# Node Command Package
A example of command line package with node using commander and colors libs

# What I Learned
* Node modules
* Commander.js and Colors.js


### Usage

How to install package
    $ npm install -g ./

List users
    $ mcp list

Find user
    $ mcp find <term>

Find user by email
    $ mcp find -e <term>

Sort users ASC
    $ mcp sort

Sort users DESC
    $ mcp sort -d
